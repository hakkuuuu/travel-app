## 🌍 Travel App

A modern web application for exploring and planning travel experiences, built with Next.js, TypeScript, and Tailwind CSS.

### 🛠 Technologies Used

- **Next.js** - A React framework for server-side rendering and static site generation.
- **TypeScript** - A statically typed language that builds on JavaScript, providing better tooling and error checking.
- **Tailwind CSS** - A utility-first CSS framework for rapidly building custom designs.
